# nyaice

serve local scratch projects.

```
Usage:
  $ nyaice [project id|project url|.sb3|.zip|project dir]

Commands:
  [project id|project url|.sb3|.zip|project dir]  project id or url or .sb3 or .zip or project dir

For more info, run any command with the `--help` flag:
  $ nyaice --help

Options:
  --port <port>  server port (default: 3000)
  --editor       open editor
  -h, --help     Display this message
  -v, --version  Display version number
```
